# WeMuduo

## 目的 
- 1、理解阻塞、非阻塞、同步、异步
- 2、理解Unix/Linux上的五种IO模型
- 3、epoll的原理以及优势
- 4、深刻理解Reactor模型
- 5、从开源C++网络库，学习优秀的代码设计
- 6、掌握基于事件驱动和事件回调的epoll+线程池面向对象编程
- 7、通过深入理解muduo源码，加深对于相关项目的深刻理解
- 8、改造muduo，不依赖boost，用C++11重构

## 编译环境
1.编译器 CLion（windows）
2.ubuntu 18.04 远程运行环境
